# AI Study Buddy

## Project Overview

AI Study Buddy is an AI-powered learning assistant developed as part of the **EDUNET FOUNDATION | IBM SkillsBuild | Artificial Intelligence | 6-Week Internship (May 2026 Batch)**.

The project is designed to help students simplify and enhance their learning experience by converting unstructured study materials into structured, easy-to-understand formats. It processes multiple input sources such as PDFs, images, and YouTube videos, and generates meaningful learning outputs using Artificial Intelligence.

---

##  Objective

The main objective of this project is to improve the way students study by using AI to automate content understanding, summarization, and revision. It reduces manual effort and helps learners focus on conceptual clarity rather than information overload.

---

##  Key Features

- **PDF Processing** – Extract and analyze text from documents  
-  **Image OCR** – Convert image-based text into editable format  
-  **YouTube Transcript Analysis** – Extract and summarize video content  
-  **AI Summarization** – Generate concise and meaningful notes  
-  **Q&A Generation** – Create question-answer pairs for better understanding  
-  **Quiz Generator** – Interactive quizzes for self-evaluation  
-  **Flashcards** – Quick revision tools for memory retention  

---

## Technologies Used

- Python  
- Streamlit  
- Natural Language Processing (NLP)  
- Optical Character Recognition (OCR)  
- AI APIs (Gemini / OpenAI or similar models)  

---

##  Working Methodology

1. User uploads study material (PDF / Image / YouTube link)  
2. System extracts raw text from the input source  
3. AI model processes and understands the content  
4. The system generates:
   - Structured summaries  
   - Question and answer sets  
   - Interactive quizzes  
   - Flashcards for revision
