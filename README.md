# AICTE-Batch1-Internship-May-2026---Ai-Powered-Study-Buddy-
AI Study Buddy is an AI-powered learning assistant developed during the EDUNET FOUNDATION | IBM SkillsBuild 6-week AI Internship (May 2026 Batch). It processes PDFs, images, and YouTube content to generate summaries, Q&amp;A, quizzes, and flashcards, helping students learn, revise, and understand concepts more efficiently.
